URL Shortener MicroService

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

  • https://excellent-tower.glitch.me/shorten/https://www.freecodecamp.org/challenges/url-shortener-microservice
  • https://excellent-tower.glitch.me/shorten/https://github.com/VeronicaM/url-shortner

Example creation output

{ "original_url":"http://foo.com:80", "short_url":" https://excellent-tower.glitch.me/8170" }

Usage:

https://excellent-tower.glitch.me/8170

Will redirect to:

https://www.google.com/
